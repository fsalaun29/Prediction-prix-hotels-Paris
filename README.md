# Projet Python pour l'économiste - 2A
## Scraping et analyse de données d'hôtels parisiens provenant du site Booking.com.
Les scripts Python sont écrits dans un fichier Jupyter Notebook.

- Le fichier "Partie 1.ipynb" est relatif au scraping des données sur Booking.com. Pour pouvoir être exécuté correctement, le fichier "chromedriver.exe" doit être placé dans le même répertoire que le code Python. Ce dernier aboutit à la création d'un data-frame regroupant un ensemble d'informations exploitables (nom de l'hôtel, prix, note moyenne des clients, nombre d'avis, arrondissement). Le data-frame est enfin sauvegardé sous la forme d'un fichier Excel ("hotels.xlsx") que l'on utilisera par la suite.
